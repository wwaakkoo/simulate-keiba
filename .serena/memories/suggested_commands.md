# Suggested Commands

## Backend (Python)
- **Run API**: `uvicorn app.main:app --reload` (inside `backend/app`)
- **Run Tests**: `python -m pytest tests/ -v`
- **Lint Check**: `python -m ruff check app/`
- **Type Check**: `python -m mypy app/`

## Frontend (TypeScript)
- **Dev Server**: `npm run dev`
- **Build**: `npm run build`
- **Run Tests**: `npm run test`
- **Lint**: `npm run lint`
- **Format**: `npm run format`

## General
- **System**: Windows
- **Commits**: Conventional Commits (`feat:`, `fix:`, etc.)
